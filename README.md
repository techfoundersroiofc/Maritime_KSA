# 📖 About the Project

The **KSA Maritime College Website** is the official digital platform developed for **KSA Maritime College**, a prestigious institution under the **KSA Group of Institutions**.

The website has been designed to provide an engaging and informative digital experience for prospective students, parents, faculty members, and stakeholders while showcasing the institution's academic excellence, maritime programs, infrastructure, placements, admissions, achievements, and campus life.

Built using modern web technologies, the application emphasizes performance, scalability, accessibility, responsive design, and SEO best practices.

# 📋 Prerequisites

Before running the project, ensure the following are installed:

- Node.js (v18 or later)
- npm (v9 or later)
- Git

Verify installation:

```bash
node -v
npm -v
git --version
```

---

# ⚙️ Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate to the project directory

```bash
cd Maritime_KSA
```

Install dependencies

```bash
npm install
```

---

# ▶️ Running the Development Server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```
