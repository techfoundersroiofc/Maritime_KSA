# 🌊 KSA Maritime College Website

A modern, responsive, and high-performance official website developed for **KSA Maritime College**, a premier institution under the **KSA Group of Institutions**.

The website is designed to provide an exceptional user experience for prospective students, parents, and stakeholders while showcasing the institution's academic excellence, maritime programs, campus facilities, admissions, and achievements.

---

## 📌 Project Overview

The KSA Maritime College website is built using modern web technologies with a focus on:

- Responsive Design
- Fast Loading Performance
- SEO-Friendly Architecture
- Clean UI/UX
- Accessibility

- Easy Content Management
- Optimized User Experience

# ⚙️ Installation

Clone the repository

```bash
git clone <repository-url>
```

Move into the project directory

```bash
cd Maritime_KSA
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open your browser

```
http://localhost:3000
```
